# **App Name**: BillEase

## Core Features:

- User Authentication: User-friendly login and signup using email/password.
- Customer Management: Add, edit, and manage customer contact information.
- Invoice Generation: Create, view, and send invoices to customers.
- Payment Tracking: Track payments and manage due invoices.
- PDF Export: Download invoices as PDFs for record-keeping.
- AI Powered Smart Summary Sales Reports: Provide a tool to offer a sales report that can be filtered and categorized. LLM decides when it should include data categories based on which have the most information to report on.
- Product/Service Management: Add and manage available products or services.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: Soft purple (#7E57C2) to highlight key actions.
- Body and headline font: 'PT Sans' (sans-serif) for a modern look and readability.
- Minimalist design with clear and intuitive navigation. Mobile-responsive layout to support access from different devices.
- Simple, flat icons to represent common actions such as "create invoice", "add customer", and "track payment."
- Subtle transitions and animations to provide visual feedback for user interactions, enhancing the overall user experience.
